# الفصل الخامس: الطباعة والتنسيق (Printing & Formatting)

## 🖨️ الدرس الثالث والعشرون: تنسيق الأرقام الصحيحة (Integer Format - printf)

في الدروس السابقة كنا نعتمد كلياً على كائن الطباعة cout لطباعة المخرجات على الشاشة. cout ممتاز وسهل، ولكنه يفتقر إلى "المرونة" عندما نريد طباعة النصوص بتنسيق معين (مثل إضافة أصفار على يسار الرقم، أو محاذاة الأرقام).

هنا يأتي دور الدالة العظيمة printf (وهي دالة موروثة من لغة C وموجودة في كل لغات البرمجة تقريباً). هذه الدالة تمنحك تحكماً كاملاً في شكل المخرجات.

1. الاستخدام الأساسي (الرمز %d):

تعتمد دالة printf على فكرة (الأماكن المحجوزة - Placeholders). الرمز %d يُخبر الكومبايلر: "احجز هذا المكان، وسأعطيك رقماً صحيحاً (Integer) لتضعه هنا".

<div dir="ltr">

```cpp
#include <iostream>
using namespace std;

int main()
{
    int page = 1;

    // cout طريقة:
    // cout << "The page number = " << page << "\n";

    // printf طريقة:
    printf("The page number = %d \n", page);

    return 0;
}
```

</div>

2. طباعة أكثر من متغير (Multiple Variables):

إذا كان لديك عدة متغيرات تريد طباعتها في نفس الجملة، كل ما عليك فعله هو إضافة أكثر من %d بالترتيب الذي تريده، ثم تمرير المتغيرات مفصولة بفاصلة ,.

<div dir="ltr">

```cpp
int page = 1;
int totalPages = 10;

// سيتم وضع المتغير الأول في %d الأولى، والثاني في %d الثانية
printf("You are in page %d of %d \n", page, totalPages);
// المخرجات: You are in page 1 of 10

```

</div>

3. العمليات الحسابية داخل الـ printf (Expressions):

لست مضطراً لتخزين نتيجة الجمع في متغير جديد لتطبعها. يمكنك إجراء العملية الحسابية مباشرة داخل المكان المخصص للمتغيرات في الدالة.

<div dir="ltr">

```cpp
int num1 = 20;
int num2 = 30;

printf("The result of %d + %d = %d \n", num1, num2, num1 + num2);
// المخرجات: The result of 20 + 30 = 50

```

</div>

4. حشو الأصفار ومحاذاة العرض (Width Specification & Zero Padding):

من أقوى ميزات printf هي قدرتها على توحيد طول الأرقام (مثلاً جعل كل الأرقام تتكون من خانتين 01, 02 .. 10).
نستخدم لذلك الصيغة %0\*d، حيث يتم إرسال رقمين: الأول يحدد العرض المطلوب (Width)، والثاني هو المتغير الفعلي.

<div dir="ltr">

```cpp
int page = 1;

// نريد الرقم أن يشغل خانتين (2)، وإذا كان أقل، املأ الفراغ بأصفار
printf("Page: %0*d \n", 2, page);
// المخرجات: Page: 01

// نريد الرقم أن يشغل 3 خانات
printf("Page: %0*d \n", 3, page);
// المخرجات: Page: 001

// نريد الرقم أن يشغل 4 خانات
printf("Page: %0*d \n", 4, page);
// المخرجات: Page: 0001

```

</div>

(ملاحظة: يمكنك أيضاً كتابة العرض مباشرة في الرمز هكذا %02d أو %03d بدون الحاجة لتمرير رقم العرض كمتغير إضافي، وسيعطي نفس النتيجة).

💡 الفائدة الهندسية:
استخدام printf يجعل الكود الخاص بك أنظف وأسهل في القراءة، خاصة عندما تكون الجملة النصية طويلة وتحتوي على متغيرات كثيرة، بدلاً من التقطيع المستمر باستخدام علامات << في الـ cout.

## 🧮 الدرس الرابع والعشرون: تنسيق الأرقام العشرية (Float Format - printf)

كما تعلمنا كيفية تنسيق الأرقام الصحيحة باستخدام %d، سنتعلم الآن كيفية التحكم الكامل في الأرقام العشرية (الكسور) ذات النوع float أو double باستخدام الرمز %f.

التركيز الأساسي هنا هو على "الدقة" (Precision)، أي: كم خانة عشرية تريد أن تظهر بعد الفاصلة؟

1. تحديد عدد الخانات العشرية (Precision):

لنفترض أن لدينا المتغير PI = 3.14159265. لطباعة عدد معين من الخانات بعد الفاصلة، نستخدم الصيغة %.xf (حيث x هو عدد الخانات).

%.1f: يطبع خانة واحدة بعد الفاصلة (3.1).

%.2f: يطبع خانتين بعد الفاصلة (3.14).

2. التقريب التلقائي (Automatic Rounding):

لغة C++ ذكية جداً في دالة printf. عندما تطلب منها قص الرقم العشري عند خانة معينة، فإنها تنظر للرقم الذي يليه، وتُطبق قوانين التقريب الرياضية (إذا كان 5 فأكثر، يتم رفع الرقم الأخير بمقدار 1).

إذا طلبنا %.3f للرقم 3.14159، المنطقي أن يطبع 3.141، لكنه في الواقع سيطبع 3.142 (لأن الرقم الرابع هو 5، فقام بتقريب الواحد إلى اثنان).

إذا طلبنا %.4f، سيطبع 3.1416 (لأن الرقم الخامس هو 9، فتم تقريب الخمسة إلى ستة).

3. حشو الأصفار (Zero Padding for Floats):

ماذا لو طلبنا طباعة 4 خانات عشرية لرقم لا يمتلك سوى خانتين؟ (مثال: 12.45).
ببساطة، سيقوم الكومبايلر بحشو الباقي أصفاراً ليحقق التنسيق المطلوب، فيطبع 12.4500.

💻 مثال عملي شامل بالكود:

<div dir="ltr">

```cpp
#include <iostream>
#include <cstdio> // يُفضل تضمين هذه المكتبة لاستخدام printf
using namespace std;

int main()
{
    float PI = 3.14159265;
    double d = 12.45;
    float x = 7.0;
    float y = 9.0;

    // 1. تحديد الدقة والتقريب
    printf("Precision 1: %.1f \n", PI); // المخرجات: 3.1
    printf("Precision 2: %.2f \n", PI); // المخرجات: 3.14
    printf("Precision 3: %.3f \n", PI); // المخرجات: 3.142 (تم التقريب)
    printf("Precision 4: %.4f \n", PI); // المخرجات: 3.1416 (تم التقريب)

    // 2. حشو الأصفار
    // طلبنا 4 خانات، والرقم به خانتين فقط
    printf("Double value is: %.4f \n", d); // المخرجات: 12.4500

    // 3. العمليات الحسابية المباشرة
    // قسمة 7 على 9 وعرض النتيجة لـ 3 خانات عشرية
    printf("The float division %.3f / %.3f = %.3f \n", x, y, x / y);
    // المخرجات: The float division 7.000 / 9.000 = 0.778

    return 0;
}

```

</div>

🚨 ملاحظة هامة جداً:
عملية القص والتقريب وحشو الأصفار التي تتم بواسطة printf هي مجرد تنسيق شكلي على الشاشة (Visual Formatting). القيمة الحقيقية المخزنة داخل المتغير PI في الذاكرة تظل كما هي بكامل دقتها (3.14159265) ولا تتغير أو تتأثر أبداً.

## 🔤 الدرس الخامس والعشرون: تنسيق النصوص والحروف (String and Char Format - printf)

الآن وقد فهمنا كيفية طباعة الأرقام بأنواعها، حان الوقت لتعلم كيفية التعامل مع النصوص (Strings) والحروف الفردية (Characters) باستخدام printf.

1. مشكلة دالة printf مع الـ std::string (تحذير هام):

لأن دالة printf هي دالة قديمة موروثة من لغة الـ C، فهي لا تدعم كائن النصوص الحديث string الخاص بـ C++ بشكل مباشر. إذا قمت بتعريف string name = "Mohammed"; وحاولت تمريره لـ printf، سيظهر لك خطأ أو تُطبع بيانات مشوهة (Garbage).

🛠️ الحل الهندسي:
يجب استخدام "مصفوفة الحروف" (C-Style Strings) بدلاً من string. نقوم بتعريف مصفوفة char فارغة الأقواس، ونخزن بداخلها النص. الرمز المستخدم لحجز مكان للنص هو %s (اختصاراً لـ String).

<div dir="ltr">

```cpp
#include <iostream>
#include <cstdio>
using namespace std;

int main()
{
// تعريف نص باستخدام مصفوفة حروف (C-Style String)
char Name[] = "Mohammed Abu-Hadhoud";
char SchoolName[] = "Programming Advices";

    // استخدام %s لطباعة النصوص
    printf("Dear %s, How are you?\n", Name);
    // المخرجات: Dear Mohammed Abu-Hadhoud, How are you?

    printf("Welcome to %s School!\n", SchoolName);
    // المخرجات: Welcome to Programming Advices School!

    return 0;
```

}

</div>

2. تنسيق الحروف الفردية (Char Format):

لطباعة حرف واحد (Character)، نستخدم الرمز %c.
ميزة قوية: مثلما قمنا بالتحكم في عرض الأرقام (Width)، يمكننا التحكم في عرض الحرف لإضافة مسافات فارغة (Spaces) قبله لغرض ترتيب المخرجات على الشاشة. نستخدم لذلك الصيغة %\*c حيث النجمة تمثل عدد الخانات الإجمالي الذي سيشغله الحرف.

<div dir="ltr">

```cpp
int main()
{
    char c = 'S';

    // طباعة الحرف كما هو
    printf("Char: %c \n", c);

    // حجز 2 خانات للحرف (سيقوم بوضع مسافة واحدة فارغة ثم الحرف)
    printf("Char: %*c \n", 2, c);

    // حجز 5 خانات (سيقوم بوضع 4 مسافات فارغة ثم الحرف)
    printf("Char: %*c \n", 5, c);

    return 0;
}

```

</div>

💡 الفائدة العملية:
التحكم في عرض الحروف (Width Setting) مفيد جداً عندما تقوم ببناء جداول (Tables) أو أعمدة نصية (Columns) على واجهة الـ Console، حيث يساعدك في محاذاة البيانات (Alignment) بشكل مثالي واحترافي لتبدو كشاشة أنظمة الطيران أو القوائم المنظمة.

## 📏 الدرس السادس والعشرون: أداة تحديد العرض (Setw Manipulator)

في كثير من الأحيان، نحتاج لطباعة البيانات على شكل جداول (Tables) منظمة لتبدو احترافية. المشكلة تظهر عندما نستخدم cout العادية لطباعة نصوص بأطوال مختلفة (مثلاً اسم كورس طويل واسم كورس آخر قصير)، مما يؤدي إلى ظهور جدول "متعرج" وغير مرتب تماماً ولا يمكن ضبطه يدوياً باستخدام المسافات Spaces.

لحل هذه المشكلة في C++، نستخدم أداة سحرية تسمى setw (اختصاراً لـ Set Width).

1. ما هي أداة setw وكيف تعمل؟

هي أداة معالجة (Manipulator) وليست دالة عادية. وظيفتها حجز "عدد معين من الخانات" للنص أو المتغير الذي سيتم طباعته بعدها مباشرة.

طريقة عملها: إذا حجزت 10 خانات مثلاً setw(10)، وطبعت كلمة تتكون من 4 حروف، ستقوم الأداة بطباعة الكلمة بمحاذاة لليمين (Right-Aligned) وتملأ الـ 6 خانات المتبقية بمسافات فارغة على اليسار، ليظل التنسيق ثابتاً مهما اختلف طول الكلمة.

المكتبة المطلوبة: لاستخدام هذه الأداة، يجب تضمين مكتبة المعالجات <iomanip> (Input/Output Manipulators).

💻 مثال عملي (رسم جدول احترافي بالكود):

<div dir="ltr">

```cpp
#include <iostream>
#include <iomanip> // خطوة ضرورية لاستخدام setw
using namespace std;

int main() 
{
    // طباعة رأس الجدول (Header)
    cout << "--------------------------------------------------\n";
    cout << setw(10) << "Code" << " | " << setw(25) << "Course Name" << " | " << setw(5) << "Mark" << "\n";
    cout << "--------------------------------------------------\n";

    // طباعة بيانات الكورسات (لاحظ كيف ستبدو الأعمدة مرتبة تماماً)
    cout << setw(10) << "C101" << " | " << setw(25) << "Intro to Programming 1" << " | " << setw(5) << "95" << "\n";
    cout << setw(10) << "C102" << " | " << setw(25) << "Computer Hardware" << " | " << setw(5) << "88" << "\n";
    cout << setw(10) << "C103" << " | " << setw(25) << "Networks" << " | " << setw(5) << "75" << "\n";
    
    cout << "--------------------------------------------------\n";

    return 0;
}

```
</div>

💡 نصيحة هندسية من المهندس أبو هدهود:
كيف تختار الرقم المناسب داخل قوس الـ setw()؟
دائماً اختر رقماً أكبر من أطول نص تتوقع طباعته في هذا العمود. مثلاً، إذا كان أطول اسم كورس في قائمتك يتكون من 20 حرفاً، اجعل عرض العمود 25 (setw(25))، لتضمن دائماً وجود مسافة فارغة تفصل بينه وبين العمود الذي يليه، فيبدو الجدول متناسقاً وجميلاً بشكل دائم.
