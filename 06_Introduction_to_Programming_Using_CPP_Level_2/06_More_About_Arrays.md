# الفصل السادس: المزيد عن المصفوفات (More About Arrays)

## 🗄️ الدرس السابع والعشرون: المصفوفات ثنائية الأبعاد (Two Dimensional Arrays)

بعد أن أتقنا التعامل مع المصفوفات أحادية البعد (One-Dimensional Arrays)، حان الوقت للانتقال لمستوى أعلى.

ما هي المصفوفة ثنائية الأبعاد؟
تخيلها ببساطة كأنها "جدول" (Table) يتكون من صفوف (Rows) و أعمدة (Columns).

بدلاً من أن يكون لديك خط واحد من الصناديق، أصبح لديك شبكة كاملة من الصناديق (مثل شيت الإكسيل).

### 1. طريقة التعريف والتهيئة (Declaration & Initialization)

لتعريف مصفوفة ثنائية، نحتاج لتحديد حجمين داخل أقواس مربعة [ ][ ].

القوس الأول: عدد الصفوف (Rows).

القوس الثاني: عدد الأعمدة (Columns).

<div dir="ltr">

```cpp
#include <iostream>
using namespace std;

int main()
{
// تعريف مصفوفة تتكون من 3 صفوف و 4 أعمدة
int x[3][4] = {
{1, 2, 3, 4}, // الصف الأول (Index 0)
{5, 6, 7, 8}, // الصف الثاني (Index 1)
{9, 10, 11, 12} // الصف الثالث (Index 2)
};

    // طباعة عنصر معين (مثلاً الرقم 7 موجود في الصف 1 والعمود 2)
    cout << x[1][2] << endl; // النتيجة: 7

    return 0;

}
```

</div>

💡 ملاحظة هامة: تماماً كالمصفوفات العادية، الفهرس (Index) يبدأ دائماً من الصفر. لذلك، إذا أردت الوصول للعمود الرابع في الصف الأول، ستكتب x[0][3]. دائماً اطرح 1 من الترتيب الذي تفكر فيه.

### 2. كيفية المرور على عناصر المصفوفة ثنائية الأبعاد؟

للمرور على مصفوفة أحادية كنا نستخدم حلقة for واحدة. أما هنا، فنحن بحاجة إلى حلقتي تكرار متداخلتين (Nested For Loops):

الحلقة الخارجية (Outer Loop): تتحكم في الصفوف.

الحلقة الداخلية (Inner Loop): تتحكم في الأعمدة.

💻 تطبيق عملي (الواجب): طباعة جدول الضرب (10 × 10)

الهدف من هذا الواجب هو إنشاء مصفوفة 10x10، وتعبئتها بنواتج جدول الضرب (من 1 إلى 10)، ثم طباعتها بشكل جدول منظم باستخدام دالة printf التي تعلمناها في فصل التنسيق.

<div dir="ltr">

```cpp
#include <iostream>
#include <cstdio> // لاستخدام printf
using namespace std;

int main()
{
    // 1. حجز مصفوفة 10 صفوف و 10 أعمدة
    int table[10][10];

    // 2. تعبئة المصفوفة بنواتج الضرب
    for (int i = 0; i < 10; i++) // حلقة الصفوف
    {
        for (int j = 0; j < 10; j++) // حلقة الأعمدة
        {
            // نزيد 1 لأن الفهارس تبدأ من صفر بينما جدول الضرب يبدأ من 1
            table[i][j] = (i + 1) * (j + 1);
        }
    }

    // 3. طباعة المصفوفة كجدول منظم
    cout << "\n--- Multiplication Table (10x10) ---\n\n";
    for (int i = 0; i < 10; i++)
    {
        for (int j = 0; j < 10; j++)
        {
            // استخدام printf لطباعة الرقم في خانتين (مثال: 01, 02 .. 10)
            printf("%02d ", table[i][j]);
        }
        cout << endl; // سطر جديد بعد كل صف
    }

    return 0;
}
```

</div>

💡 الفائدة الهندسية (Debugging):
المصفوفات ثنائية الأبعاد مفيدة جداً في برمجة الألعاب (مثل الشطرنج أو الـ Tic-Tac-Toe)، وكذلك في معالجة الصور (Image Processing) حيث تُمثل كل خلية "بكسل" في الشاشة له إحداثيات (X و Y).
